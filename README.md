# ShortBot
A bot created for the Short Gang's discord server. Made for fun with no expectations. Anyone is welcome to contribute :)

## Setup
[Git](https://git-scm.com/) is highly recommended to install and publish changes. You can either:

* Clone the repository with Git into your desired folder
* Download ZIP and unzip in your desired folder

## Installation
Requires [Python](https://www.python.org/) installed in PATH (preferably the latest version. Minimum 3.10).

### For Windows (run line by line in your terminal):

#### 0. Navigate to the bot's folder
```
cd <path-to-folder>
```
Replace `<path-to-folder>` with the actual path

#### 1. Create a virtual environment (recommended)
```
python -m venv .venv
.venv\Scripts\activate
```

#### 2. Install dependencies
```
pip install .
```
